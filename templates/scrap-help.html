  • Cache options
    • Disabled
      An HTTP request is sent to retrieve the page to scrap, the page is not cached.
    • Enabled
      The page is retrieved from the cache if it has already been cached. If the page is not present in the cache, an HTTP request is sent to retrieve the page which is then cached.
    • Force Update
      An HTTP request is sent to retrieve the page to scrap, even if it is already present in the cache. The page is then cached.
